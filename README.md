# DreamID-V-Pinokio

A Pinokio launcher for [DreamID-V](https://github.com/manat0912/DreamID-V-Pinokio.git)

## Features

- 1-Click Installation
- Git LFS support for large models
- Automatic model linking to Pinokio shared drive
- Gradio Web UI

## How to use

1. Install [Pinokio](https://pinokio.computer/)
2. Open Pinokio and search for DreamID-V-Pinokio or click Install.
3. Run the "Install" script.
4. Run the "Start" script.
